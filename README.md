### 🚀 Featured Projects

#### 🔹 Automated Economic Releases — [automated-economic-releases](https://www.linkedin.com/in/mateus-restier/details/projects/) 🔒
> Automated bulletin generation for economic indicators (INCC-M, IGP-M, IGP-DI, ICOMEX): Excel ingestion, LLM-driven narrative writing, and Word (.docx) rendering via docxtpl — with a Streamlit UI for operator input.

#### 🔹 Document RAG System — [document-vector-pipeline](https://www.linkedin.com/in/mateus-restier/details/projects/) · [document-rag-framework](https://www.linkedin.com/in/mateus-restier/details/projects/) 🔒
> End-to-end semantic search and RAG system for Portuguese-language journalism and economic articles. The pipeline ingests from MongoDB, handles noisy informal text with spaCy lemmatization and NLTK stopword removal, and stores dense embeddings in Qdrant (streaming mode, AWS DocumentDB compatible). A LangGraph-orchestrated RAG framework then enriches economic analysis text with contextual information retrieved from Qdrant, using LLM-based relevance validation, multi-query generation, configurable quality thresholds, and Azure OpenAI integration. Built for AWS SageMaker deployment.

#### 🔹 COPOM RAG System — [copom-vector-pipeline](https://github.com/MateusRestier/copom-vector-pipeline) · [copom-rag-api](https://github.com/MateusRestier/copom-rag-api) · [copom-streamlit](https://github.com/MateusRestier/copom-streamlit) 🔓
> 🌐 [Live app](https://copom-rag.streamlit.app/) — End-to-end RAG system for question answering over COPOM (Brazilian Monetary Policy Committee) documents. The pipeline downloads meeting minutes and policy communications from the BCB Open Data API, parses PDFs, and stores Google Gemini embeddings in PostgreSQL + pgvector. A FastAPI layer retrieves relevant chunks via cosine search and generates answers with Gemini. A Streamlit app provides the Q&A interface and an admin panel for document ingestion.

#### 🔹 Nightsight Analytics — [nightsight-analytics](https://github.com/MateusRestier/nightsight-analytics) 🔓
> Economic activity monitor derived from VIIRS nighttime satellite imagery. Ingests and processes NASA/NOAA radiance data via Google Earth Engine, loads it into Snowflake, and visualizes regional luminosity trends through an interactive Streamlit + PyDeck dashboard.

#### 🔹 Insight Invest — [insight-invest](https://github.com/MateusRestier/insight-invest) 🔓
> End-to-end automated stock analysis, forecasting, and recommendation system using web scraping, RandomForest models, PostgreSQL, and an interactive Dash/Plotly dashboard.

#### 🔹 Sound DNA — [sound-dna](https://github.com/MateusRestier/sound-dna) 🔓
> End-to-end pipeline for music genre classification: YouTube ingestion (yt-dlp), 369 audio features via DSP (librosa), ML models (XGBoost/Random Forest), and a Streamlit app with interactive spectral analysis + genre prediction.

#### 🔹 Scheduled Action Bot — [scheduled-action-bot](https://www.linkedin.com/in/mateus-restier/details/projects/) 🔒
> Human-in-the-loop web automation bot: scheduled triggers send an email confirmation request; upon user approval via IMAP reply, a Selenium agent executes a browser workflow with randomized timing. Deployed on Render (Docker) with GitHub Actions keepalive and cron-job.org for precise scheduling.

#### 🔹 Competitor Analysis — [competitor-analysis](https://www.linkedin.com/in/mateus-restier/details/projects/) 🔒
> Competitor analysis automation pipeline: web scraping, data consolidation, and structured reporting to support strategic pricing and market positioning decisions.

#### 🔹 Football DataOps Lakehouse — [football-dataops-lakehouse](https://github.com/MateusRestier/football-dataops-lakehouse) 🔓
> Local football data lakehouse simulating AWS architecture (S3 → Athena → MWAA) using MinIO, DuckDB, Dagster, and Great Expectations — ingesting StatsBomb open data through a medallion pipeline (raw → validated → trusted).

#### 🔹 JoyBind — [joybind](https://github.com/MateusRestier/joybind) 🔓
> Maps controller buttons to custom keyboard strokes and absolute screen coordinates. Built with Python to simplify and automate macro interactions in games.
